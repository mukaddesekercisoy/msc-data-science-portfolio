# IJC437 – Introduction to Data Science
## Coursework

This repository contains coursework completed for the module **IJC437 – Introduction to Data Science** as part of the MSc Data Science programme at the University of Sheffield.

## Project overview
The coursework focuses on fundamental data science skills, including data exploration, basic data cleaning, and analysis using R.

## Data
The project uses structured datasets suitable for introductory data science analysis.

## Methods
All analysis is conducted in R, primarily using tidyverse-based workflows.

## Outputs
The outputs include summary statistics, visualisations, and written interpretation of results.
